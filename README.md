# 🏢 Smart Meeting Room Booking System with AI

## 📌 Project Description

The Smart Meeting Room Booking System is a web-based application developed using TypeScript, Node.js, Express.js, MySQL, HTML5, CSS3, Bootstrap 5, and JavaScript.

This system helps organizations manage meeting room reservations efficiently. Users can book, edit, and cancel meeting room reservations through a user-friendly interface. The system also includes an AI Assistant that provides booking notifications, detects duplicate reservations, and automatically updates meeting statuses based on the scheduled time.

## ⚙️ Installation

```bash

```

```bash
npx tsc
```

### Run the application

```bash
npm start
```

---

## 🗄️ Database Setup (phpMyAdmin)

1. Start **Apache** and **MySQL** using XAMPP.
2. Open **phpMyAdmin**.

```
http://localhost/phpmyadmin
```

3. Create a new database named:

```text
meeting_room
```

4. Select the **meeting_room** database.
5. Click **Import**.
6. Import the following file:

```text
sql/database.sql
```

7. Click **Import** to create the required database tables.
